# BASIC CODES


### cd
```bash
cd File/
```
> Goes to the directory.

> Dizine gider.

----------------------------------------

### ls
```bash
ls
```
> Lists the contents of the array.

> Dizinin içeriğini listeler.



```bash
ls -la
```
> Shows hidden files in a list.

> Gizli dosyaları liste halinde gösterir.

----------------------------------------

### pwd
```bash
pwd
```
> Shows the directory.

> Dizini gösterir.

----------------------------------------

### mkdir
```bash
mkdir NewFile
```
> Creates a new file.

> Yeni dosya oluşturur.

----------------------------------------

### rm
```bash
rm 123.txt
```
> Creates a new file.

> Yeni dosya oluşturur.
